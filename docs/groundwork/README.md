# Groundwork Pack (Vector Grounding Reply)

This folder tracks claim-level evidence and drafting constraints.

## Files
- `01-citation-ledger.md`: source-level citation log from local PDFs.
- `02-quote-bank.md`: verified verbatim quotes with page anchors.
- `03-definitions-sheet.md`: fixed term definitions for consistency.
- `04-contribution-boundary.md`: scope split vs the bridge paper.
- `05-objection-matrix.md`: objections, strongest sources, and replies.
- `06-claim-evidence-matrix.md`: section-by-section claim support.
- `07-venue-constraints.md`: venue fit and submission requirements.
- `08-structured-abstract.md`: one-page abstract scaffold.
- `09-original-docs-claim-audit.md`: automated audit of source-note claims.

## Workflow
1. Finalize definitions in `03`.
2. Fill `01` and `02` with page-level anchors.
3. Use `05` and `06` before drafting each section.
4. Keep `09` updated after adding or editing notes.
